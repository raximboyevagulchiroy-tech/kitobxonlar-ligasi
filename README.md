# 📚 Kitobxonlar Ligasi — Veb Ilovasi

"Kitobxonlar Ligasi" — o'zbek adabiyoti ixlosmandlari uchun maxsus yaratilgan interaktiv va ma'rifiy veb-platformadir. Ushbu platforma kitobxonlar o'rtasida mutolaa madaniyatini yuksaltirish, kitoblar haqida fikr almashish va olingan bilimlarni testlar orqali sinovdan o'tkazishga xizmat qiladi.

---

## ✨ Imkoniyatlari va Asosiy funksiyalari

* 🔐 **Xavfsiz Ro'yxatdan O'tish:** Ilovadan to'liq foydalanish uchun foydalanuvchilar avval ro'yxatdan o'tadilar (Sign Up) va o'z profillariga kirishadi (Log In).
* 📖 **O'zbek Adabiyoti Ro'yxati:** O'zbek adabiyotining eng sara durdonalari jamlangan boy kitoblar bazasi.
* 💬 **Kitobxonlar Sharhlari:** Har bir kitob sahifasida boshqa kitobxonlar tomonidan qoldirilgan taqriz va sharhlarni o'qish hamda o'z fikrini qoldirish imkoniyati.
* 📝 **Interaktiv Testlar Tizimi:** Mutolaa qilingan asarni qay darajada tushunganini tekshirish uchun har bir kitobga oid **50 tadan maxsus test** yechish imkoniyati.
* 📥 **PDF Yuklab Olish:** Kitoblarni elektron shaklda (PDF formatida) to'g'ridan-to'g'ri qurilmaga yuklab olish funksiyasi.
* 🛒 **Xarid Uchun Bog'lanish:** Agar foydalanuvchi kitobning bosma (qog'oz) shaklini sotib olmoqchi bo'lsa, tezkor bog'lanish tugmasi orqali buyurtma berish imkoniyati.
* 🏆 **Kitobxonlar Reytingi:** Test natijalari va faollikka qarab foydalanuvchilar o'rtasida shakllanadigan "Liga" reyting tizimi.

## 🛠 Ishlatilgan Texnologiyalar (Tech Stack)

* **Backend:** Python, Django Freymvorki (MVC arxitekturasi)
* **Frontend:** HTML5, CSS3, JavaScript (Dinamik testlar va interfeys elementlari uchun)
* **Ma'lumotlar bazasi:** SQLite
* **Server (Deployment):** PythonAnywhere
